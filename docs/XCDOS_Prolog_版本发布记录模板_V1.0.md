## 变更类型说明

- **feat**: 新功能
- **fix**: Bug 修复
- **perf**: 性能优化
- **refactor**: 重构（无功能变更）
- **chore**: 依赖/配置/工具变更
- **hotfix**: 紧急修复（线上问题）
